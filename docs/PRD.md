# 소모임 이벤트 관리 MVP PRD

## 핵심 정보

**목적**: 수영/헬스/탁구 등 소모임 주최자가 공지, 참여자 관리, 정산 등 반복 업무를 카카오톡·단체 채팅 없이 한 곳에서 처리한다.
**사용자**: 정기 소모임을 운영하는 주최자(organizer)와 해당 모임에 참여하는 멤버(member).

---

## 사용자 여정

```
1. 로그인/회원가입 페이지
   ↓ 인증 완료 (이메일 또는 Google OAuth)

2. 대시보드 페이지
   ↓ "팀 만들기" 클릭 (첫 방문 또는 신규 팀 필요 시)
   ↓ 기존 팀 있으면 → 팀 홈 페이지로 이동

3. 팀 생성 페이지
   ↓ 팀 정보 입력 후 생성 완료

4. 팀 홈 페이지
   ↓ [주최자] "이벤트 만들기" 클릭
   ↓ [멤버]   공유된 이벤트 링크로 직접 진입 가능

5. 이벤트 생성 페이지
   ↓ 이벤트 정보 입력 후 저장

6. 이벤트 상세 페이지
   ↓ [멤버]    "참여 신청" 클릭 → 신청 완료(정원 내) 또는 대기 등록
   ↓ [주최자]  "공유 링크 복사" → 카카오톡 등으로 전달
   ↓ [주최자]  "참여자 관리" 클릭 → 참여자 관리 페이지로 이동
   ↓ [주최자]  "정산 생성" 클릭 → 정산 생성 페이지로 이동

   [비로그인 사용자] 이벤트 공개 공유 페이지 진입
   → 이벤트 내용 조회만 가능
   → 로그인 후 참여 신청 가능

7. 참여자 관리 페이지
   ↓ 대기자 확정/취소 처리 완료 → 이벤트 상세 페이지로 복귀

8. 정산 생성 페이지
   ↓ 총비용 입력 → 1/N 자동 계산 → 정산 생성 완료

9. 정산 현황 페이지
   ↓ 납부 확인 처리 → 완료 상태 변경

10. 팀 초대 수락 페이지 (/join/[token])
    ↓ 비로그인 → 로그인 페이지로 리디렉션 후 복귀
    ↓ 로그인 상태 → 팀 멤버로 합류 → 대시보드 페이지로 이동
```

## 기능 명세

| -------- | ---------------------- | --------------------------------------------- | --------------------------------- | ---------------------------------------------------- |

| **F005** | 참여 신청/취소         | 정원 내 즉시 확정, 정원 초과 시 대기 등록     | 오프라인 단체 채팅 의존도 제거    | 이벤트 상세 페이지, 이벤트 공개 공유 페이지          |
| **F006** | 대기자 자동 승격       | 확정자 취소 시 대기 1번 자동 확정 처리        | 주최자 수동 관리 부담 제거        | 이벤트 상세 페이지, 참여자 관리 페이지               |
| **F007** | 참여자 관리 (주최자)   | 주최자가 참여자 확정/취소/대기 상태 수동 조정 | 주최자 통제권 제공                | 참여자 관리 페이지                                   |

| **F009** | 정산 현황 및 납부 확인 | 미납/납부 현황 표시, 주최자가 납부 확인 처리  | 납부 독촉 수작업 제거             | 정산 현황 페이지                                     |
| **F010** | 대시보드               | 내가 속한 팀 목록 + 다가오는 이벤트 요약      | 진입점 역할, 팀/이벤트 전환 허브  | 대시보드 페이지                                      |
| **F012** | 팀 설정                | 팀 정보 수정, 종목 변경 (주최자 전용)         | 팀 운영 기본 관리                 | 팀 설정 페이지                                       |

## 메뉴 구조

```
[공개 메뉴 — 비로그인]
├── 이벤트 공개 공유 페이지 (/events/[id])
│   └── 기능: F004 (이벤트 내용 조회), F005 (로그인 후 참여 신청 유도)
├── 팀 초대 수락 페이지 (/join/[token])
│   └── 기능: F002 (초대 링크 합류) — 비로그인 시 로그인 페이지로 리디렉션
├── 로그인 페이지
│   └── 기능: F011 (이메일/Google OAuth 로그인)
└── 회원가입 페이지
    └── 기능: F011 (이메일 회원가입)

[로그인 후 — 공통 상단 내비게이션]
├── 대시보드
│   └── 기능: F010 (내 팀 목록 + 다가오는 이벤트)
└── 팀 만들기
    └── 기능: F001 (팀 생성)

[팀 내부 메뉴 — 팀 선택 후]
├── 팀 홈
│   └── 기능: F003 (이벤트 목록 조회 및 이벤트 생성 진입)
├── 멤버 관리
│   └── 기능: F002 (초대 링크 생성/갱신), F007 (멤버 역할 조회)
├── 팀 설정 (organizer 전용)
│   └── 기능: F012 (팀 정보 수정)
└── 이벤트 목록 → 이벤트 상세
    ├── 기능: F005 (참여 신청/취소), F006 (대기자 승격 확인)
    ├── 참여자 관리 (organizer 전용)
    │   └── 기능: F006 (대기자 자동 승격), F007 (상태 수동 조정)
    ├── 이벤트 수정 (organizer 전용)
    │   └── 기능: F003 (이벤트 정보 수정)
    └── 정산
        ├── 정산 현황
        │   └── 기능: F009 (납부 현황 조회 및 확인 처리)
        └── 정산 생성 (organizer 전용)
            └── 기능: F008 (1/N 자동 분배)
```

## 페이지별 상세 기능

### 대시보드 페이지

> **구현 기능:** `F010` | **메뉴 위치:** 로그인 후 공통 상단 내비게이션

| 항목            | 내용                                                                                                                                                                                                 |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 서비스 진입점 — 내 팀 목록과 다가오는 이벤트를 한눈에 파악하는 허브                                                                                                                                  |
| **진입 경로**   | 로그인 완료 후 자동 리디렉션, 상단 내비게이션 "대시보드" 클릭                                                                                                                                        |
| **사용자 행동** | 내가 속한 팀 카드 확인 후 팀 홈으로 이동, 다가오는 이벤트 카드 클릭                                                                                                                                  |
| **주요 기능**   | • 내가 속한 팀 목록 카드 (팀명, 종목, 내 역할 표시)<br>• 다가오는 이벤트 요약 리스트 (날짜, 팀명, 참여 상태)<br>• **"팀 만들기"** 버튼 (팀 생성 페이지로 이동)<br>• 팀 없을 때 온보딩 안내 문구 표시 |
| **다음 이동**   | 팀 카드 클릭 → 팀 홈 페이지, 이벤트 카드 클릭 → 이벤트 상세 페이지, "팀 만들기" → 팀 생성 페이지                                                                                                     |

### 팀 생성 페이지

> **구현 기능:** `F001` | **메뉴 위치:** 대시보드 "팀 만들기" 버튼

| 항목            | 내용                                                                                                                                                                   |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 새로운 소모임 팀 공간을 생성하는 폼 페이지                                                                                                                             |
| **진입 경로**   | 대시보드 "팀 만들기" 버튼 클릭                                                                                                                                         |
| **사용자 행동** | 팀 이름, 설명, 종목(sport_type) 입력 후 생성                                                                                                                           |
| **주요 기능**   | • 팀 이름 입력 (필수, 최대 50자)<br>• 팀 설명 입력 (선택)<br>• 종목 선택 (수영/헬스/탁구/기타 등 셀렉트)<br>• 생성 시 invite_token 자동 발급<br>• **"팀 만들기"** 버튼 |
| **다음 이동**   | 성공 → 팀 홈 페이지, 실패 → 에러 메시지 인라인 표시                                                                                                                    |

### 팀 홈 페이지

> **구현 기능:** `F003`, `F010` | **메뉴 위치:** 팀 내부 메뉴 "팀 홈"

| 항목            | 내용                                                                                                                                                                                 |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **역할**        | 팀 단위 이벤트 목록 조회 및 이벤트 생성 진입점                                                                                                                                       |
| **진입 경로**   | 대시보드 팀 카드 클릭, 팀 내부 내비게이션 "팀 홈" 클릭                                                                                                                               |
| **사용자 행동** | 이벤트 목록 스크롤, 이벤트 카드 클릭으로 상세 진입, 주최자는 이벤트 생성                                                                                                             |
| **주요 기능**   | • 이벤트 목록 (상태별 탭: 예정/완료/취소)<br>• 이벤트 카드 (제목, 날짜, 장소, 참여 현황 n/max, 상태 배지)<br>• 팀명 및 종목 헤더 표시<br>• **"이벤트 만들기"** 버튼 (organizer 전용) |
| **다음 이동**   | 이벤트 카드 클릭 → 이벤트 상세 페이지, "이벤트 만들기" → 이벤트 생성 페이지                                                                                                          |

| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

## 구현 단계

### Phase 1: 팀 + 이벤트 기반 구조

- `teams`, `team_members`, `events` 테이블 생성 + RLS 정책
- 대시보드 페이지, 팀 생성 페이지, 팀 홈 페이지
- 이벤트 생성 페이지, 이벤트 상세 페이지, 이벤트 수정 페이지
- 이벤트 공개 공유 페이지 (`/events/[eventId]`)
- **완료 기준**: 팀 생성 → 이벤트 생성 → 공유 링크 전달 가능

### Phase 2: 참여 신청 + 멤버 관리

- `event_participants` 테이블 생성 + RLS 정책
- 참여 신청/취소/대기자 자동 승격 로직
- 참여자 관리 페이지 (organizer)
- 팀 멤버 관리 페이지, 팀 초대 수락 페이지 (`/join/[token]`)
- 팀 설정 페이지
- **완료 기준**: 공유 링크 → 신청 → 대기 → 주최자 확정 흐름 완성

### Phase 3: 정산

- `settlements`, `settlement_items` 테이블 생성 + RLS 정책
- 정산 생성 페이지 (1/N 자동 분배, 나머지 주최자 합산)
- 정산 현황 페이지 (납부 확인 + 완료 처리)
- **완료 기준**: 정산 생성 → 개인 금액 표시 → 납부 완료 처리 동작
