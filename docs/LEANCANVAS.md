## 1. 문제 (Problem)

> 소모임 운영 시 반복적으로 겪는 핵심 불편

**주요 문제**

- 이벤트 공지가 카카오톡 단체 채팅방에 묻혀 찾기 어렵고 누락 발생
- 참여 신청·취소 상태를 주최자가 직접 수동으로 취합·추적해야 함
- 정원 초과 시 대기자 관리를 주최자가 일일이 수작업으로 처리
- 정산 계산 및 납부 독촉이 모두 수작업이라 분쟁 발생

**기존 대안**

- 카카오톡 단체 채팅 + 댓글 참여 확인
- 구글 폼으로 신청받고 별도 스프레드시트로 정산

---

## 2. 고객 세그먼트 (Customer Segments)

**Primary — 소모임 주최자 (Organizer)**

- 10~30인 규모의 정기 운동 모임 운영자 (수영·헬스·탁구 등)
- 매주 또는 격주로 반복되는 이벤트를 직접 기획·공지하는 사람
- 참가비 정산, 인원 관리에 시간을 쓰고 싶지 않은 사람

**Secondary — 소모임 참여자 (Member)**

- 주최자가 공유한 링크로 이벤트를 확인하고 참여 신청하는 사람
- 개인 납부 금액을 투명하게 확인하고 싶은 사람

**얼리 어답터**

- 정기 운동 소모임을 카카오톡으로만 운영하다가 관리 피로도를 느낀 주최자

---

## 3. 가치 제안 (Unique Value Proposition)

**핵심 메시지**

> "단체 채팅 없이, 링크 하나로 소모임을 운영하세요"

**주최자에게**

- 이벤트 공지·참여자 관리·정산을 한 곳에서 처리
- 대기자 자동 승격으로 수동 관리 부담 제거
- 카카오톡 공유 링크 한 번으로 신청 접수 완료

**참여자에게**

- 로그인 없이도 이벤트 내용 확인 가능
- 참여 신청·취소를 직접 처리 (주최자에게 따로 연락 불필요)
- 정산 금액과 납부 현황을 투명하게 확인

---

## 4. 해결책 (Solution)

**F003 — 이벤트 생성 및 공개 공유**

- 날짜·장소·인원·참가비 입력으로 이벤트 공지 생성
- 비로그인 사용자도 열람 가능한 공개 공유 링크 자동 발급

**F005 · F006 — 참여 신청 및 대기 자동 관리**

- 정원 내 즉시 확정, 정원 초과 시 자동 대기 등록
- 확정자 취소 시 대기 1번 자동 승격 처리

**F008 · F009 — 정산 자동화**

- 총비용 입력 → 확정 참여자 수 기준 1/N 자동 분배
- 개인별 납부 현황 및 주최자 납부 확인 처리

**F002 — 팀 초대 링크**

- 토큰 기반 초대 URL로 멤버 모집
- 기존 링크 무효화 및 재발급 기능

---

## 5. 경쟁우위 (Unfair Advantage)

**쉽게 복사할 수 없는 차별점**

- **소모임 특화 UX** — 대규모 행사 플랫폼이 아닌 10~30인 정기 소모임에 최적화된 단순한 흐름
- **비로그인 이벤트 조회** — 카카오톡 링크를 받은 누구나 바로 이벤트 확인 가능 (가입 장벽 최소화)
- **카카오톡 공유 → 원클릭 참여** — 기존 소통 채널(카카오톡)을 대체하지 않고 보완
- **대기자 자동 승격** — 주최자 개입 없이 자동으로 인원이 채워지는 운영 자동화

---

## 6. 수익 흐름 (Revenue Stream)

**MVP 단계 (현재)**

- 무료 — 사용자 확보 및 핵심 가치 검증 우선

**향후 수익 모델 (검토)**

- 프리미엄 플랜 — 팀 수 제한 해제, 이벤트 히스토리 무제한, 반복 이벤트 자동 생성
- 정산 수수료 모델 — 정산 기능 사용 시 소액 수수료

---

## 7. 비용 구조 (Cost Structure)

**인프라 비용**

- Supabase — DB·인증·스토리지 호스팅 (Free tier → Pro $25/월)
- Vercel — Next.js 배포 (Hobby 무료 → Pro $20/월)

**개발 비용**

- 초기 MVP 개발 인건비 (1인 개발)
- 유지보수 및 기능 확장 비용

---

## 8. 주요 지표 (Key Metrics)

**획득 지표**

- 신규 팀 생성 수 (주/월)
- 초대 링크 → 팀 합류 전환율

**활성화 지표**

- 팀 생성 후 첫 이벤트 생성까지의 시간
- 이벤트 공유 링크 → 참여 신청 전환율

**유지 지표**

- 월간 활성 주최자 수 (MAO)
- 주최자 1인당 월평균 이벤트 생성 수

**수익 지표 (향후)**

- 정산 기능 사용률
- 프리미엄 전환율

---

## 9. 채널 (Channels)

**바이럴 채널 (핵심)**

- 카카오톡 이벤트 공유 링크 — 참여자가 링크를 받아 서비스를 자연스럽게 접하는 주요 진입점
- 팀 초대 링크 — 멤버가 주변에 소모임을 소개하며 신규 팀 생성 유도

**직접 채널**

- 구글·네이버 검색 (SEO) — "소모임 관리", "운동 모임 정산" 키워드
- 운동 커뮤니티 및 SNS 홍보 (인스타그램, 러닝 오픈채팅 등)
